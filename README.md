# <a name='installation-guide'>Installation Guide</a>

This project requires the following tools:

- [Java or JDK](https://www.oracle.com/java/technologies/downloads/#java7-windows) - Java 7 or JDK 1.7.
- [NetBeans](https://netbeans-ide.informer.com/8.0/) - NetBeans 7 or 8.

To get started, install Java or JDK and Netbeans on your local computer if you don't have them already.

## Getting Started

**Step 1. Clone the code into a fresh folder**

```
git clone https://github.com/thisismemukul/java-mysql-bakery-n-cafe.git
cd java-mysql-bakery-n-cafe
```

**Step 2. Open Project in Netbeans.**
